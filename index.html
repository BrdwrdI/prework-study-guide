HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • What are the roles of the head and meta elements?
    • The head element is a container for metadata.
    • meta elements contain information about the page that is used by the browser.
    • The user won’t see information in the head or meta elements.

CSS

  • Cascading Style Sheets (CSS): programming language to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS:
    • inline CSS
      • When a developer wants to style an element directly in the HTML file
    • internal CSS style sheet
      • An Internal Style sheet is when you embed an entire stylesheet directly into the HTML file using a stlyle element in the head element
    • external CSS style sheet
      • An external CSS style sheet allows developers to keep all their CSS rules in a separate file
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • A 'String' is a sequnce of text. A string is enclosed in single or double quote marks
  • An 'Array' is a structure that allows you to store multiple values in a single reference
  • An 'Object' can be anything. Everthing in JavaScript is an object and can be stored as a variable
  • Note that whereas the style.css file is linked within the head element, the script.js file is linked within the body element.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
    • JavaScript is read from top to bottom, and order counts.
    • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • A 'for' loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
    • A 'for' loop contains the following three important statements
      • The first statement determines the starting point for our loop.
        • var x = 0;
      • The second statement is the condition.
        • x < shapes.length;
      • The final statement is what allows the array to iterate over each item.
        • x++
    • A 'for' loop starts with 'for'. So far our code looks like:
      • for(var x = 0; x < shapes.length; x++) { // code block }
    • Finally, we put the code that we want to be executed for each loop inside curly brackets
  • A 'function' is simply a set of instructions that tells the computer how to perform a certain task.
